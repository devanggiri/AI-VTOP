#NAVIGATE INTO AI-VTOP
->Step1
Login page opens
->Step2
Enter your credentials if valid 
->Step3
Enter(BAI), valid only for (BAI)
->Step4
If Not valid, Access denied
->Step5
Main Page Opens up
->Step6
Different menus can be accessed from side bar
->Step7
View your Courses, My info, Credits earned/remaining, etc.



#ACCESS COURSE TABLES
->Step1
Login to AI-VTOP website
->Step2
Go to Main Page
->Step3
Go to side Menu bar(at left)
->Step4
Select Courses from side Menu
->Step5
Choose your desired course from the section
->Step6
The selected course table opens up
->Step7
Similarly navigate to other courses via menu.
